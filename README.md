
# Interactive Code Butterfly Visualization

프루티거 에어로 감성의 나비 날개를 코드·모션·사운드로 실험할 수 있는 웹앱입니다. 원본 Figma 프로토타입(https://www.figma.com/design/r1O1WRbDLJAxZUwyqQlNUu/Interactive-Code-Butterfly-Visualization)을 로컬에서 실행하고 수정할 수 있도록 정리했습니다.
